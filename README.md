## Описание

Проект для автоматического выполнения HTTP-запросов к сервису Яндекс.Игры с ограничением на количество запросов.

## Функциональность

Скрипт `index.php` выполняет следующие действия:

1. Создает bash-скрипт `rqscr.sh` для выполнения HTTP-запросов
2. Настраивает cron-задачу для выполнения скрипта каждую минуту
3. Ведет логирование отправленных запросов в файл `log.txt`
4. Ограничивает количество запросов до 10 штук

## Принцип работы

- Bash-скрипт проверяет количество записей "Запрос отправлен" в лог-файле
- Если количество меньше 10, выполняется curl-запрос к `https://yandex.ru/games`
- Каждый успешный запрос логируется
- Cron-задача запускает скрипт ежеминутно

## Файлы проекта

- `index.php` - основной PHP-скрипт для настройки и запуска
- `rqscr.sh` - генерируемый bash-скрипт (создается автоматически)
- `log.txt` - файл логирования запросов (создается автоматически)

## Требования

- PHP с поддержкой функций `file_put_contents()`, `chmod()`, `exec()`
- Доступ к командной оболочке для настройки cron
- Утилита `curl` для выполнения HTTP-запросов

## Использование

1. Запустите PHP-скрипт:
   ```bash
   php index.php
   ```

2. Скрипт автоматически создаст bash-файл и добавит задание в cron

3. Для проверки статуса cron-задач используйте:
   ```bash
   crontab -l
   ```

4. Наслаждайтесь. Кушайте и не обляпайтесь.
